# 실행 가이드
**Requirements**
* Node.js
* NPM

**Installation**
```
git clone https://github.com/dongbin420/fe-sprint-coz-shopping
cd fe-sprint-coz-shopping
npm install
```

**Start**
```
npm run dev
```
